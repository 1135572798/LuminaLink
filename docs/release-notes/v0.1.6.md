# LuminaLink v0.1.6

This patch improves translation provider setup and removes confusing duplicated category filters.

## Changes

- Add DeepSeek translation provider preset.
- Add provider defaults for model, Base URL, and API key environment source.
- Auto-save translation provider settings to `%APPDATA%/LuminaLink/config.json`.
- Rehydrate saved settings when reopening the Settings view.
- Hide global asset type tabs inside dedicated Agent and Other Files views.
- Rename the sidebar Agent entry from "项目 Agent" to "Agent".
- Add DeepSeek support to the generated Agent helper script.

## QA

- `pnpm build`
- CLI DeepSeek default persistence test with temporary APPDATA/LOCALAPPDATA.
- Electron smoke screenshots for Settings, Agent, and Other Files views.
