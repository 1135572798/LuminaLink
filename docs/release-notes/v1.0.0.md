# LuminaLink v1.0.0 正式版

这是 LuminaLink 的第一个正式版，重点补齐大量本机资产场景下的筛选、批量翻译、跳过翻译和中文阅读体验。

## 新增与优化

- 新增按需多选模式：正常浏览不显示复选框，需要批量操作时再进入多选。
- 新增选中项批量翻译，只处理未翻译、失败或已过期的资产。
- 新增跳过翻译，跳过后的资产不再出现在未翻译队列，也不会继续消耗翻译额度。
- 新增源文件更新检测：已翻译文件发生变化后标记为“需更新”，保留旧中文译文，可手动重新翻译。
- 资产列表优先展示中文摘要，帮助用户直接判断 skill、plugin、Agent 是否有用。
- 新增列表描述悬停预览，可查看被截断的完整摘要。
- 新增右键菜单：翻译此项、阅读窗口、打开文件、定位文件、跳过翻译、复制路径。
- 优化按钮 hover、focus、active、运行中和完成反馈，刷新、扫描、翻译、跳过等操作会即时更新左侧数量。
- 设置页改为单栏页面，移除右侧资产详情和翻译面板，避免误解。
- 保留 OpenAI、DeepSeek、OpenAI-compatible、本地模型和 Agent 导出/导入翻译流程。

## 修复

- 修复部分列表图标大小不一致的问题。
- 修复设置页填写后再次进入显示不稳定的问题。
- 修复已翻译资产在列表中仍显示英文摘要的问题。
- 修复文件变化后旧译文状态无法区分的问题。
- 修复跳过翻译与扫描更新互相覆盖的问题。
- 修复复制路径在窗口失焦时可能受浏览器剪贴板限制影响的问题。
