# LuminaLink v0.1.8

This patch improves long-form translation reading and live translation feedback.

## Changes

- Normalize escaped `\n` / `\r\n` / `\t` sequences into readable line breaks for source and translated content.
- Normalize newly generated translation text before writing it to the local cache.
- Add a resizable right detail panel with a draggable divider.
- Persist the detail panel width in browser local storage.
- Add an independent reading window for translated text, source text, and metadata.
- Add live translation progress events so the detail panel can show streaming API output while a translation is being generated.
- Add smoke-test metrics for the detail divider, reader button, readable text state, and reader-window opening.

## QA

- `pnpm build`
- Electron UI smoke screenshots for detail panel, reader button, and no escaped newline display.
- Packaged app smoke test with reader-window opening and layout metrics.
