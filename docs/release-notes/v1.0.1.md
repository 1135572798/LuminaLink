# LuminaLink v1.0.1

这是 v1.0.0 后的交互修复版。

## 修复

- 修复资产列表描述悬停时同时出现系统原生提示框和 LuminaLink 中文说明卡的问题。
- 现在列表摘要只保留 LuminaLink 自定义中文悬浮说明，阅读时不会再出现两个提示框互相遮挡。

## 验证

- `pnpm build`
- `git diff --check`
- Windows 免安装包 smoke test。
- Smoke test 已确认 `.asset-copy` 不再携带原生 `title` 属性，自定义中文说明卡仍正常显示。
