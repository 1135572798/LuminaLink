## Fixes

- Replaced the packaged preload script with a CommonJS `.cjs` preload so
  `window.lumina` is reliably injected in installed/portable Windows builds.
- Fixed "Agent жүӢеҶҢз”ҹжҲҗеӨұиҙҘпјҡеҪ“еүҚдёҚжҳҜ Electron жЎҢйқўиҝҗиЎҢзҺҜеўғ" in the installed app.
- Fixed "жү«жҸҸеӨұиҙҘпјҡе®ўжҲ·з«Ҝ IPC жңӘиҝ”еӣһжү«жҸҸз»“жһң" in the installed app.
- The "Codex еҚҸеҠ©" page now always shows a copyable default prompt immediately.
  Runtime generation only upgrades the prompt with exact local paths.
- Added a packaged smoke-test mode to verify `status`, `scan`, and
  `agentGuide` through the real renderer bridge before release.
- Added packaged smoke-test screenshots for scan feedback and the Codex assist
  page.
