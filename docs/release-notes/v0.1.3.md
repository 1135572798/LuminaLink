## Fixes

- Added `%USERPROFILE%/.codex/agents` as a default scan root.
- Existing user configs are now automatically backfilled with missing default
  scan roots, so users do not need to delete `config.json`.
- Added `.toml` parsing for Codex Agent definitions.
- Fixed config loading when a JSON file has a UTF-8 BOM.

## Translation

- Added CLI export/import commands for agent-assisted translation:
  - `pnpm luminalink translate export-task "<output-json>" --limit 10`
  - `pnpm luminalink translate import-result "<output-json>"`
- The existing OpenAI/OpenAI-compatible Provider path remains available for
  users who want one-click translation inside the client.
