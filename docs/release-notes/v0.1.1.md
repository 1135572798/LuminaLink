# LuminaLink v0.1.1

This patch release fixes first-run usability problems reported from the Windows
installer.

## Downloads

- `LuminaLink-Setup-0.1.1-x64.exe`: Windows installer
- `LuminaLink-Portable-0.1.1-x64.exe`: Windows portable app

## Fixes

- Added visible scan progress, success, warning, and error messages.
- Fixed the scan result message being immediately overwritten by the generic ready state.
- Added an empty-state panel that explains what "扫描资产" does and how to add a directory.
- Added a dedicated "Codex 协助" page.
- The app now generates a local Agent runbook at `%APPDATA%/LuminaLink/AGENT_RUNBOOK.md`.
- The app now generates a copyable Codex prompt at `%APPDATA%/LuminaLink/CODEX_PROMPT.txt`.
- The app now generates a helper script at `%APPDATA%/LuminaLink/LuminaLink-Agent.ps1`.
- Clarified that scanning does not require a translation provider.

## Notes

- If Provider is configured as `env:OPENAI_API_KEY`, a desktop-launched app only
  sees user or machine environment variables. A key that only exists in one
  terminal session may not be visible to the installed app.
- The installer is unsigned, so Windows may show a SmartScreen warning on first
  launch.
