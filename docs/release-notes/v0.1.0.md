## Highlights

- Chinese-first desktop UI for local AI assistant assets.
- Scans Codex skills, plugins, user agent skills, and project Agent files.
- Supports adding any Markdown / text file into custom categories.
- Caches Chinese translations by content hash so repeated files only need one translation.
- Provides a CLI for Codex or other agents to scan, search, translate, and migrate data.
- Stores local config and caches outside the repository.

## Notes

- Translation is not automatic by default. Configure OpenAI or an OpenAI-compatible provider first.
- The installer is unsigned, so Windows may show a SmartScreen warning on first launch.
- Local data is stored under `%APPDATA%/LuminaLink` and `%LOCALAPPDATA%/LuminaLink`.
