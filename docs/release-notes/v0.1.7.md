# LuminaLink v0.1.7

This patch polishes the Settings and category navigation experience, and fixes a bottom status-bar layout regression.

## Changes

- Change the Settings API Key field into a password-style local secret input with an eye toggle.
- Allow direct pasted API keys as local-only translator credentials while still supporting `env:*` values.
- Redact direct API keys in generated Agent runbook/helper output.
- Add sidebar entries and counts for Skill, Plugin, Agent, Other Files, and Untranslated.
- Keep top asset type tabs only inside the full Asset Library view.
- Fix the bottom status/log bar so it stays pinned when no top notice is visible.
- Expand Electron smoke screenshots to cover the new sidebar categories and no-notice footer state.

## QA

- `pnpm build`
- Electron UI smoke test with screenshots for scan result, no-notice footer, detail actions, Skill, Plugin, Agent, Other Files, Untranslated, Settings, and Codex Assist views.
- Local config persistence and direct-key redaction checks.
