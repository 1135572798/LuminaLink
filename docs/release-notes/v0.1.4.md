# LuminaLink v0.1.4

This patch makes the Windows desktop shell feel less like a web page and keeps
asset actions visible without scrolling the whole window.

## Downloads

- `LuminaLink-Setup-0.1.4-x64.exe`: Windows installer
- `LuminaLink-Portable-0.1.4-x64.exe`: Windows portable app

## Fixes

- Removed the native File/Edit/View/Window/Help menu bar.
- Switched the window to a hidden titlebar overlay so the top frame is cleaner.
- Locked the outer app shell to the viewport and moved scrolling into the
  relevant internal panes.
- Kept the right-side asset action buttons visible at the bottom of the detail
  pane.
- Restyled internal scrollbars to match the LuminaLink UI.
